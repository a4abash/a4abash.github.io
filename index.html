Few concept about Django

Inorder to install Django on your computer - [pip install django] Make sure you install the recent stable version. After successful installation of django start the project with the command. - [django-admin startproject projectName] for e.g [django-admin startproject collegeManagementSystem] . After the creation of project make an app if you need - [python manage.py startapp appname] for e.g [django-admin startapp teacher]. Run the server to see if your project runs - [python manage.py runserver] Inorder to create database of your models you need to make the migrations.So here is the command 1. [python manage.py makemigrations] 2. [python manage.py migrate] Inorder to create the admin||SuperUser- [python manage.py createsuperuser]
